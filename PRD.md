# PRD: 교회 재정집행 캘린더 자동화 시스템

## 문서 정보
- **프로젝트명**: 재정집행일정 PPT 자동화 템플릿
- **버전**: 1.0
- **작성일**: 2026-01-08
- **대상 연도**: 2026년
- **문서 유형**: Product Requirements Document

---

## 1. Executive Summary

### 1.1 프로젝트 개요
교회 재정팀의 예산집행 일정을 자동으로 계산하고 시각적으로 표현하는 PPT 템플릿 자동 생성 시스템입니다. 규정에 따라 집행일, 결재일, 운영위원회 일정을 자동 계산하여 연간 12개월 캘린더를 생성합니다.

### 1.2 비즈니스 목표
- 수작업으로 작성하던 월별 재정 캘린더 제작 시간 **95% 절감** (12시간 → 30분)
- 규정 준수율 **100% 달성** (자동 계산으로 인적 오류 제거)
- 재정팀 업무 효율성 향상 및 표준화된 디자인 통일성 확보

### 1.3 핵심 가치 제안
- ✅ **자동화**: 규정에 따라 날짜를 자동으로 계산
- ✅ **일관성**: 모든 월에 동일한 디자인 적용
- ✅ **정확성**: 인적 오류 제거 및 규정 100% 준수
- ✅ **효율성**: 12개월 캘린더를 단 한 번의 실행으로 생성

## 3. 문제 정의

### 3.1 현재 문제점
1. **수작업 비효율성**
   - 월별 캘린더 수동 작성 시 월 1시간 소요 (연간 12시간)
   - 날짜 계산 오류 발생 가능성

2. **규정 준수 어려움**
   - 둘째주/넷째주 화요일, 전주 토요일 등 복잡한 규정
   - 수동 계산 시 실수 발생

3. **디자인 일관성 부족**
   - 매월 다른 디자인으로 제작될 수 있음
   - 색상, 폰트, 레이아웃 불일치

4. **시간 소모**
   - PPT 수동 제작 및 날짜 입력 반복 작업

### 3.2 해결 방안
규정을 코드화하여 자동으로 날짜를 계산하고, 통일된 디자인 템플릿으로 PPT를 생성하는 시스템 구축

## 8. 규정 및 정책

### 8.1 재정 집행 규정

#### 규정 1: 예산집행일
- **날짜**: 매월 둘째주/넷째주 화요일
- **계산 로직**:
  ```python
  1. 해당 월의 모든 화요일 추출
  2. 두 번째 화요일 = 둘째주 집행일
  3. 네 번째 화요일 = 넷째주 집행일
  ```

#### 규정 2: 결재 마감일
- **날짜**: 각 집행일 전주 토요일 자정까지
- **계산 로직**:
  ```python
  결재일 = 집행일 - 3일
  (화요일 기준 3일 전 = 토요일)
  ```

#### 규정 3: 운영위원회의
- **날짜**: 매월 마지막주 전주 일요일
- **계산 로직**:
  ```python
  1. 해당 월의 마지막 주 확인
  2. 그 전주 일요일 선택
  ```

### 8.2 디자인 표준
- 모든 슬라이드 동일한 색상 팔레트 사용
- NanumSquare Bold 폰트 통일
- 표 테두리 파란색 (#0061FF, 1.5pt)
