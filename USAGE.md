# 🗓️ 재정집행일정 캘린더 사용방법

교회 재정팀을 위한 예산집행일정 캘린더입니다. 월별로 자동 계산된 일정을 확인할 수 있습니다.

---

## 📌 시작하기

### 1️⃣ 웹사이트 접속
```
https://lazygass.github.io/church-accounting-team-calendar-maker/
```

### 2️⃣ 페이지 로드
- 현재 연도의 12개월 캘린더가 자동으로 표시됩니다
- 연도를 선택하면 해당 연도의 캘린더를 볼 수 있습니다

---

## 🎯 주요 기능

### 1. 연도 선택
- 상단의 **연도 드롭다운**에서 원하는 연도 선택
- 기본값: 현재 연도 (자동 감지)
- 범위: 현재 연도 ± 5년

### 2. 월별 캘린더 조회
- 1월부터 12월까지 모든 월의 캘린더 표시
- 각 캘린더는 다음 정보를 포함합니다:
  - **예산집행일**: 파란색 표시 (둘째주/넷째주 화요일)
  - **결재일**: 분홍색 표시 (예산집행일 전주 금요일~토요일)
  - **운영위원회의**: 노란색 표시 (마지막주 전주 일요일)

### 3. 이미지 다운로드
- 각 월 캘린더 하단의 **"월 이미지 다운로드"** 버튼 클릭
- PNG 형식으로 고해상도 이미지 다운로드 가능
- 파일명: `{연도}_{월}_예산집행캘린더.png`
  - 예: `2026_03_예산집행캘린더.png`

---

## 📋 일정 해석 가이드

### 예산집행일 (파란색)
- **표시 월**: 둘째주 화요일, 넷째주 화요일
- **의미**: 해당 날짜에 예산이 집행되는 날
- **조건**: 전주 토요일 자정까지 결재된 건에 한해 집행

### 결재일 (분홍색)
- **표시 월**: 예산집행일 전주 금요일 ~ 토요일
- **의미**: 예산 결재가 진행되는 기간

### 운영위원회의 (노란색)
- **표시 월**: 마지막주 전주 일요일
- **특이사항**: 2026년 3월, 4월은 마지막주 일요일

## 💡 사용 예시

### 예시 1) 2026년 3월 일정 확인
1. 웹사이트 접속
2. 연도 기본값이 **2026년**으로 설정됨
3. **3월 캘린더** 확인
   - 예산집행일: 3월 10일(둘째주 화), 24일(넷째주 화)
   - 결재일: 3월 6~7일(금~토), 20~21일(금~토)
   - 운영위원회의: 3월 29일(마지막주 일요일 - 특이)

### 예시 2) 2027년 캘린더 다운로드
1. **연도 드롭다운**에서 **2027** 선택
2. 원하는 월을 찾아 **"월 이미지 다운로드"** 버튼 클릭
3. 컴퓨터에 PNG 파일 저장

### 예시 3) 인쇄 용도
1. 원하는 월의 이미지 다운로드
2. 이미지를 인쇄 또는 SNS/메일로 공유

## ⚙️ 기술 정보

### 자동 계산 로직
- **현재 연도**: JavaScript의 `new Date().getFullYear()`로 자동 감지
- **예산집행일**: 특정 규칙에 따라 자동 계산
- **배포**: GitHub Pages (자동 갱신)

### 브라우저 호환성
- Chrome (권장)
- Firefox
- Safari
- Edge

## 🔍 문제 해결

### Q. 캘린더가 로드되지 않습니다
**A.** 브라우저 캐시 삭제 후 새로고침하세요.
- Windows/Linux: `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

### Q. 이미지 다운로드가 안 됩니다
**A.** 다음을 확인하세요:
- 브라우저에서 다운로드 허용 설정 확인
- 팝업 차단 해제
- 네트워크 연결 확인

### Q. 2027년 이후의 캘린더를 보고 싶습니다
**A.** 연도 드롭다운에서 선택 가능합니다.
- 기본 범위: 현재 연도 ± 5년
- 예: 2026년 접속 시 2025~2031년 선택 가능
