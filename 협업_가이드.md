# 다음에 더 쉽게 협업하는 방법 💡

## 1. 시각 자료 먼저 공유하기

### 좋은 예시
```
"이 PPT처럼 만들어주세요" + PPT 파일 첨부
"이 이미지처럼" + 스크린샷
"이 사이트 참고" + URL
```

### 효과
- 말로 설명하는 시간 90% 단축
- 오해 가능성 최소화
- 한 번에 정확한 결과

## 2. 피그마/파워포인트 활용

### 방법
1. 피그마나 PPT로 레이아웃 그리기
2. 위치, 크기, 색상 표시
3. "이대로 웹으로 만들어주세요"

### 장점
- 픽셀 단위 정확도
- 색상 코드 자동 추출
- 반복 수정 최소화

## 3. 단계별 접근

### 추천 순서
1. **큰 틀**: "전체 레이아웃 이렇게"
2. **확인**: 큰 틀 OK 받기
3. **세부**: "여기 색상 변경", "저기 크기 조정"

### 이번 프로젝트에서
- ❌ 한 번에 모든 요구사항
- ✅ 표 → 테두리 → 박스 → 위치 순서로

## 4. 용어 통일

### 명확한 표현
- "위로" → "상단 여백 5px"
- "크게" → "폰트 24px"
- "파란색" → "#0061FF"
- "오른쪽" → "right: 0"

## 5. 참고 자료

### 유용한 도구
- **색상**: [Coolors.co](https://coolors.co)
- **레이아웃**: Figma, Canva
- **폰트**: Google Fonts
- **스크린샷**: Windows 캡처 도구

## 다음 프로젝트 시작 템플릿

```markdown
# 프로젝트 요청

## 참고 이미지
[이미지 첨부]

## 주요 기능
1. 기능 A
2. 기능 B

## 디자인 요구사항
- 색상: #XXXXXX
- 폰트: 나눔스퀘어
- 크기: 800x600

## 우선순위
1순위: 레이아웃
2순위: 색상
3순위: 세부 조정
```

이렇게 하면 훨씬 빠르고 정확하게 만들 수 있습니다! 🚀
