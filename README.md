# 재정집행일정 캘린더 - 웹 버전

## 소개
교회 재정집행일정을 자동으로 계산하여 캘린더 형식으로 표시하고, 각 월별 이미지를 다운로드할 수 있는 웹 애플리케이션입니다.

## 주요 기능
- ✅ 자동 일정 계산
  - 예산집행일: 둘째주/넷째주 화요일
  - 결재일: 각 집행일 전주 금요일, 토요일
  - 운영위원회의: 마지막주 전주 일요일
- ✅ 12개월 캘린더 자동 생성
- ✅ 월별 이미지 다운로드 (PNG)
- ✅ 카카오톡 공유 가능한 고해상도 이미지

## 사용 방법

### 1. 브라우저에서 열기
`index.html` 파일을 더블클릭하거나 브라우저로 드래그하여 엽니다.

### 2. 연도 선택
상단의 연도 선택 드롭다운에서 원하는 연도를 선택합니다 (기본: 2026).

### 3. 캘린더 생성
"생성" 버튼을 클릭하면 12개월 캘린더가 자동으로 생성됩니다.

### 4. 이미지 다운로드
각 월 캘린더 하단의 "이미지 다운로드" 버튼을 클릭하면 PNG 이미지로 저장됩니다.

### 5. 카카오톡 공유
다운로드한 이미지를 카카오톡에 첨부하여 공유할 수 있습니다.

- Chrome, Edge, Firefox, Safari 최신 버전
- JavaScript 활성화 필요
- 인터넷 연결 필요 (html2canvas 라이브러리 로드)

## 기술 스택
- HTML5
- CSS3
- Vanilla JavaScript
- html2canvas (이미지 변환 라이브러리)

## 색상 가이드
- 파란색 (#0061FF): 예산집행일, 테두리
- 분홍색 (#FD79A8): 결재일, 일요일
- 황금색 (#FFD700): 운영위원회의
- 배경색 (#F4F8FB): 연한 파란색

## 문제 해결

### 이미지가 다운로드되지 않는 경우
- 브라우저의 팝업 차단을 해제하세요
- 다른 브라우저에서 시도해보세요

### 캘린더가 표시되지 않는 경우
- 브라우저 콘솔(F12)에서 오류 메시지를 확인하세요
- 인터넷 연결을 확인하세요 (html2canvas 로드 필요)
