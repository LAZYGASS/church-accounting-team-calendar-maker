## 주요 변경 사항

### 1. 폰트 설정
- **변경 전**: 나눔고딕 Bold
- **변경 후**: NanumSquare Bold (나눔 스퀘어 볼드)
- **적용 위치**: 모든 텍스트 (제목, 본문, 표 등)

### 2. 요일 표시
- **형식**: 영어 약자 (Sun, Mon, Tue, Wed, Thu, Fri, Sat)
- **색상**:
  - 일요일 (Sun): 핑크색 (#FD79A8)
  - 평일 (Mon-Fri): 검정색
  - 토요일 (Sat): 검정색 (#000000)

### 3. 날짜 색상
- **일요일**: 핑크색 (#FD79A8)
- **평일**: 검정색
- **토요일**: 파란색 (#4472C4)
- **집행일**: 파란색 (#0061FF) - 굵게
- **결재일**: 요일별 색상 적용 (분홍/파란색 제거)
- **지난달 날짜**: #C0D3DE (회색빛 파란색)

### 4. 표 테두리
- **색상**: 파란색 (#0061FF)
- **두께**: 1.5pt (19050 EMU)
- **적용**: 모든 셀 (요일 헤더 행 포함)

### 5. 예산집행일 규정 변경
- **변경 전**: 둘째주/넷째주 월요일
- **변경 후**: 둘째주/넷째주 화요일
- **결재일**: 각 집행일 전주 토요일 (화요일 기준 3일 전)

### 6. 예산집행일 도형
- **위치**: 각 예산집행일 날짜 셀 바로 아래
- **형식**: 도형과 텍스트 통합 (하나의 둥근 사각형)
- **색상**: 파란색 (#0061FF)
- **텍스트**: "예산집행일" (흰색, 12pt, 굵게)
- **크기**: 1.17" × 0.35"
- **배치**: 날짜 셀 중앙 정렬, 셀 하단에서 0.40" 위로 올린 위치
- **개수**: 2개 (둘째주, 넷째주 각각)
- **설명 텍스트**: 박스 옆에 "<-전 주 토요일 자정까지 결재 난 건에 한해" (검정색, 14pt, 굵게)

### 7. 결재일 도형
- **위치**: 각 결재일 날짜(토요일) 셀 바로 아래
- **형식**: 도형과 텍스트 통합 (하나의 둥근 사각형)
- **색상**: 분홍색 (#FD79A8)
- **텍스트**: "결재일" (흰색, 12pt, 굵게)
- **크기**: 금요일-토요일 2칸 너비 × 0.35" 높이
- **배치**: 금요일부터 토요일 2칸에 걸쳐 배치, 셀 하단에서 0.40" 위로
- **개수**: 2개 (각 집행일 전주 토요일)

### 8. 운영위원회 표시
- **위치**: 마지막주 전주 일요일
- **표시 형식**: 날짜 아래 "운영위원회"
- **날짜**: 20pt (요일별 색상 적용)
- **운영위원회 텍스트**:
  - 폰트 크기: 14pt
  - 글자 색상: 검정색 (#000000)
  - 강조 배경: 황금색 (#FFD700)

### 9. 헤더 날짜 형식
- **변경 전**: "예산집행캘린더 2026.01"
- **변경 후**: "예산집행캘린더 26.01"
- **이유**: 텍스트 박스 내 잘림 방지

### 10. 지난달 날짜 표시
- **위치**: 첫 번째 주의 빈 칸
- **색상**: #C0D3DE (연한 회색-파란색)
- **계산 방식**: 이전 달 마지막 날짜부터 역순으로 표시

### 11. 배경색
- **슬라이드 배경**: #F4F8FB (연한 파란색)
- **표 셀 배경**: 흰색

## 일정 계산 로직

### 예산집행일 (둘째주/넷째주 화요일)
```python
# 화요일 (인덱스 2) 찾기
for week in calendar:
    tuesday = week[2]
    if tuesday != 0:
        tuesday_weeks.append(tuesday)

# 둘째주, 넷째주 선택
execution_days = [tuesday_weeks[1], tuesday_weeks[3]]
```

### 결재일 (전주 토요일)
```python
# 집행일로부터 3일 전 (화요일 -> 토요일)
approval_date = execution_date - timedelta(days=3)
```

### 운영위원회의 (마지막주 전주 일요일)
```python
# 마지막 주 찾기
last_week = calendar[-1]
# 전주 일요일
prev_week = calendar[-2]
committee_day = prev_week[0]  # 일요일
```

## 주요 함수

### `calculate_schedule(year, month)`
- 규정에 따른 일정 자동 계산
- 반환: 집행일, 결재일, 운영위원회의 날짜

### `create_calendar_slide(year, month, ...)`
- 캘린더 슬라이드 생성
- 자동/수동 모드 지원

### `_add_month_number(slide, month)`
- 왼쪽 상단 큰 월 숫자 추가 (파란색, 72pt)

### `_add_header_box(slide, year, month)`
- 헤더 박스 추가 ("예산집행캘린더 26.01")

### `_add_calendar_table(slide, ...)`
- 캘린더 표 생성
- 요일, 날짜, 색상, 테두리 설정

### `_add_legend(slide, ...)`
- 예산집행일 도형 추가
- 결재일 박스 추가

## 참고 사항
- 표 테두리는 EMU 단위 사용 (12700 = 1pt)
- 텍스트 수직 정렬: `vertical_anchor = 1` (중앙)
- 둥근 모서리 사각형: shape type = 5
- RGB 색상: 튜플로 저장 후 `RGBColor(*tuple)` 방식으로 사용
- 테두리 색상: 튜플을 직접 받아 16진수로 변환
